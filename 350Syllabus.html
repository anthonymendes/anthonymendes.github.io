Textbook

There is no official textbook for the course. For those wanting a second reference (beyond googling), a reasonable textbook is Mathematica: A problem centered approach, 1st edition by Roozbeh Hazrat. This text can be downloaded at no cost from the "Springerlink" database on the Cal Poly library web site.

Course content

This course will consist of two acts. Act I will introduce the basics of Mathematica programming. Act 2 will explore topics in various mathematics courses (such as Calculus and Differential equations, Linear Algebra, Combinatorics and Graph Theory, Number theory, etc.) using Mathematica.

The official department course outline contains details on the course content. Upon completion of this course, students will be able to

  1. Understand how to use mathematical software to perform calculations pertaining to calculus, linear algebra and numerical analysis.
  2. Write and debug programs using the Mathematica programming language.
  3. Construct graphical output from the mathematical software.
