Content and learning objectives

I have been greatly looking forward to sharing the ideas in this course with you! Upon completion of this course, students will be able to

  1. finding generating functions for combinatorial sequences,
  2. use the singularities of these generating functions to find the asymptotic growth rate of sequences,
  3. become familiar with fundamental combinatorial structures such as set partitions, integer partitions, permutations, and tableaux, and
  4. gain an advanced understanding of proof techniques in discrete mathematics, including proof by bijection and sign-reversing involutions. The official department course outline contains more details.

Although the topics in this course are outstanding, my primary objective is to train you to think like a discrete mathematician.

Undergraduate MATH 435 vs. Graduate MATH 530

The course content for the both courses is the same, but the assignments are not. The graduate assignments may assume knowledge of material from the undergraduate curriculum (i.e., concepts from abstract algebra, linear algebra, complex analysis, real analysis, and PDE's) that cannot be assumed for the undergraduate assignments. A graduate student without this background should be able to learn what is needed independently to complete the exercises.

Exercise Sets

This course is designed for the mathematically mature. Expect to spend significant time on the sometimes challenging exercises. Please get started on the exercises early!

Follow these steps when turning in the assignments and exams:

  1. Neatly and clearly write out the solutions by hand. This may require writing rough drafts; please do not submit messy scratch work. If you are using a tablet to write out solutions or know mathematical typesetting (LaTeX) or have any other method of creating legible solutions in a .pdf file, then of course this can be used instead of handwritten solutions.
  2. It is difficult to read and grade .jpg image files, so please create a single .pdf file from your exercise sets using a document scanner phone application.
  3. Upload the file to in our Canvas site before the due date.

Late assignments will be penalized 20% per day late. If life issues get in the way, please let me know before the assignment is due so that we can make special arrangements.

Sribes

Each class session will have an assigned scribe. The scribe will write detailed lecture notes from the days class and then email me a .pdf file with the filename name "DiscreteNotes-09-21.pdf" (with 09-21 changed to the class date) to upload onto our web site with 48 hours. The notes will be graded on correctness and legibility. Please try to limit the file size to at most 1MB.
