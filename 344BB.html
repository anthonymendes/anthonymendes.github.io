Blue Book Exercises

Blue book exercises will be updated here after they are given in class.

  1. Find \(\displaystyle\mathcal{L} \left[ 1 + 2 \sin^2(3 t) + t^4 \right]\).
  2. Find \(\displaystyle\mathcal{L}^{-1} \left[ \frac{4 s - 2}{s^2+4} + \frac{4}{s} \right]\).
  3. Using the Laplace transform, solve \(\begin{cases} y'' - y' = -t \\ y(0) = 0, y'(0) = 1. \end{cases}\)
  4. Find \(\displaystyle \mathcal{L} \left[ e^{-2 t} \cos 4 t \right]\) and \(\displaystyle \mathcal{L}^{-1} \left[ \frac{s}{(s+2)^2 + 9} \right]\).
  5. Find \(\displaystyle \mathcal{L} \left[ u_{2 \pi}(t) \sin 2 t \right]\) and \(\displaystyle \mathcal{L}^{-1} \left[ e^{-3 s}/(s^2 + 4) \right]\)
  6. Solve \(y'(t) = g(t)\) with \(y(0) = 1\) where \(g(t)\) is the function which is \(2-t\) for \(t < 2\) and \(0\) otherwise.
  7. Solve \(\displaystyle \begin{cases} y' - y = \delta(t - 3), \\ y(0) = 24. \end{cases}\)
  8. Find \(\displaystyle \mathcal{L}^{-1} \left[ \dfrac{1}{s+2} \cdot \dfrac{1}{s+1} \right]\) using convolution.
  9. Solve \(x^2 y'' - x y' - 3 y = 0\).
  10. Solve \(x^2 y'' + 3 x y' + 5 y = 0\).
  11. Solve \(y' + 6 y = e^{-2x}\) using the integrating factor.
  12. Solve \(y'' - x y' - y = 0\).
  13. Solve \(2 x^2 y'' - x y' + (1 - x)y = 0\).
